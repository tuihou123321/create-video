# 🚀 快速启动指南

## 欢迎使用AI视频生成器！

这是一个全新架构的应用，包含精美的Landing Page和强大的视频生成工具。

## ⚡ 立即开始

### 1. 启动后端服务
打开终端，运行：
```bash
cd proxy-server
npm start
```

✅ 后端服务将运行在 `http://localhost:3001`

### 2. 启动前端应用
新开一个终端，运行：
```bash
cd video-site
npm start
```

✅ 前端应用将自动打开浏览器，访问 `http://localhost:3000`

## 🎬 使用流程

### 步骤1: 浏览Landing Page
首次访问会看到精美的Landing Page，包含：
- 🎯 产品介绍
- ✨ 核心功能展示
- 💡 使用优势说明
- ❓ 常见问题解答

### 步骤2: 进入工具页面
点击 **"免费开始创作"** 按钮，进入视频生成器

### 步骤3: 配置视频参数
在配置页面设置：
- 📝 **基本设置**: 文本内容、配音选择、角色图片
- 🎨 **视觉设置**: 背景图片、水印、标题文字
- 🎵 **音频设置**: 背景音乐、音量调节
- ✨ **样式设置**: 字幕样式、背景处理方式

### 步骤4: 生成视频
点击 **"生成视频"** 按钮，系统将：
1. 文本转语音（TTS）
2. 语音转字幕（ASR）
3. AI生成配图
4. 智能背景处理
5. 合成视频预览

### 步骤5: 预览和调整
- ▶️ 播放预览生成的视频
- 🔄 如需调整，点击"重新配置"
- 🏠 点击"返回首页"回到Landing Page

## 🐛 常见问题

### Q: 端口被占用怎么办？
**A:** 更换端口或关闭占用端口的程序：
```bash
# 查看占用情况
lsof -i :3000
lsof -i :3001

# 或在package.json中修改端口
```

### Q: 样式不生效？
**A:** 尝试硬刷新（Ctrl+Shift+R）清除缓存

### Q: 如何跳过Landing Page？
**A:** 在 `App.tsx` 中修改初始状态：
```typescript
const [currentPage, setCurrentPage] = useState<'landing' | 'generator'>('generator');
```

### Q: 生成失败怎么办？
**A:** 检查：
1. 后端服务是否运行
2. API密钥是否配置
3. 网络连接是否正常
4. 浏览器控制台错误信息
